How It Works
Trust Architecture
01
Pyth Oracle Layer
High-frequency price feeds from 90+ publishers, aggregated with confidence intervals. Streamed via Hermes SSE and verifiable on Cardano via Pyth pull oracle contracts.
02
KERI Identity Anchor
Sessions bound via WebAuthn passkeys. Every query signed against a KERI AID. Delegated agent AIDs operate under scoped, time-limited authority from the human conductor.
03
Cardano Settlement
ACDC credentials anchored via Blockfrost. On-chain attestation of data consumption events via ixn interaction events, creating an immutable audit trail on Cardano.
